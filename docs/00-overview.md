## 一句话定位

一个**理解式**英语学习 Agent：不死记硬背，而是在语境中理解；练习时分"练习模式（即时纠错）"与"考试模式（延迟纠错 + 雅思/托福打分）"两档；生词与错误自动沉淀成个人画像，越用越懂你。

## 三大功能（概览）

1. **生词收集**：输入英文 → 切词 → 逐词问认识与否 → 不认识的连同来源句存入生词本。
2. **话题练习**：给定话题，四种模式 —— a) 引导写作 b) 引导口语 c) 自由写作打分 d) 对话打分。考试模式延迟纠错，结束后统一复盘。
3. **理解式背单词**：a) 逐词背，用户说出自己的理解，agent 判断；b) 用生词造短文，用户翻译，在语境中记忆。

## 数据闭环（设计核心）

```
功能1 产出生词 ──────────────→ 功能3 消费背诵
功能2 产出错误 ──→ 错题本 ──→ 复盘报告
所有数据 ──→ 用户画像（水平基线、遗忘曲线、错误模式）──→ 反哺所有功能
```

## 关键设定（已定）

- **开源项目**，本地优先，**无账号系统**（单用户，schema 预留 `user_id`）。
- 后端 **Python (FastAPI)**，前端 **React/TS**。
- **含语音**（STT + TTS）纳入 MVP。
- 一切外部依赖（存储 / LLM / STT / TTS / 发音评估）皆**适配器**，可换云服务或自部署。
